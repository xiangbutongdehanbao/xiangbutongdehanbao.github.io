使用Hexo+GitHub搭建自己的个人博客超详细

Created2022-07-03Updated2022-07-03

简介

自己的博客是在b站上:https://www.bilibili.com/video/BV1mU4y1j72n?spm_id_from=333.1007.top_right_bar_window_custom_collection.content.click&vd_source=f390bf48284d686096e85919d080875e 这个网址所学的这是2021版的，如果有最新版按自己的最新版操作，2021版以前由于Github两次更新不能在使用否则会报错。

报错处理

本人在搭建博客的时候只有一处报错：

在最后上传Github时出现：

hexo d输入后显示lenovo@LAPTOP-VR7JVP69 MINGW64 /e/BLOG
$ hexo d
INFO Validating config
You should configure deployment settings in _config.yml first!

Available deployer plugins:
git

For more help, you can check the online docs: https://hexo.io/

这应该是前面输入：

deploy:

type: git

repository:https://github.com/你的名字/仓库的名字.git

branch: 分支

出现错误，仔细检查有没有少输入空格或者标点符号。hexo d 命令输入后有其他报错是因为科学网速慢，一定要多试几次 一定 一定！！！因为我每次上传GitHub都回失败几次都是因为网速不行。
